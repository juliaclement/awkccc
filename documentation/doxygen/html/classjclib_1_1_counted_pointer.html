Detailed Description

template<typename T>
class jclib::CountedPointer< T >

This is an implementation of an intrusive counted pointed. The typename T must expose two public methods CountedPointerAtttach and CountedPointerDetach. The expected behaviour is that these will maintain a count of attached pointers and self delete when the counter reaches zero
